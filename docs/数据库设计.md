# 运城果业记账库存管理系统 - 数据库设计

## 一、数据库概述

- **数据库名称**：fruit_management

- **排序规则**：utf8mb4_unicode_ci

## 三、ER图关系说明

```
t_user (用户表)
  ├── 1:N ── t_customer (客户表)
  ├── 1:N ── t_product (果品表)
  ├── 1:N ── t_inbound_order (入库单表)
  ├── 1:N ── t_outbound_order (出库单表)
  ├── 1:N ── t_inventory (库存表)
  └── 1:N ── t_debt (欠款表)

t_customer (客户表)
  ├── 1:N ── t_outbound_order (出库单表)
  └── 1:N ── t_debt (欠款表)

t_product (果品表)
  ├── 1:N ── t_inbound_order (入库单表)
  ├── 1:N ── t_outbound_order (出库单表)
  └── 1:1 ── t_inventory (库存表)

t_inbound_order (入库单表)
  └── N:1 ── t_product (果品表)

t_outbound_order (出库单表)
  ├── N:1 ── t_customer (客户表)
  ├── N:1 ── t_product (果品表)
  └── 1:0..1 ── t_debt (欠款表)

t_debt (欠款表)
  └── N:1 ── t_customer (客户表)
```

## 六、密码说明

系统使用BCrypt加密算法进行密码加密。

**测试账号：**
- 用户名：`admin`
- 密码：`123456`

如需生成新的BCrypt密码哈希，可以使用项目中的 `PasswordGenerator.java` 工具类，或使用以下Java代码：

```java
import org.springframework.security.crypto.bcrypt.BCryptPasswordEncoder;

public class PasswordGenerator {
    public static void main(String[] args) {
        BCryptPasswordEncoder encoder = new BCryptPasswordEncoder();
        String rawPassword = "你的密码";
        String encodedPassword = encoder.encode(rawPassword);
        System.out.println("加密后的密码：" + encodedPassword);
    }
}
```
