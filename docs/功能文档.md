# 运城果业记账库存管理系统 - 完整功能文档

## 一、系统定位

专为运城果商、冷库、代办、合作社打造的手机管理系统
解决：记账乱、库存不清、欠款对不上、客户管理难的问题。

## 二、适用对象

- 水果收购代办
- 冷库老板
- 果品批发商
- 小型合作社
- 经常送货、赊账的果商

## 三、核心功能模块

### 1. 入库管理（进货）
- **录入信息**：果品名称、品种、等级、重量、单价、金额、产地、入库时间
- **支持功能**：一车货、多批次录入
- **自动计算**：总金额、总重量
- **记录信息**：入库人、备注

### 2. 出库管理（卖货）
- **选择信息**：选择客户、选择货品、出库重量、单价、金额
- **支持功能**：现款 / 欠款
- **自动计算**：利润、库存扣减
- **生成单据**：出货单

### 3. 欠款管理（最核心痛点）
- **欠款信息**：谁欠、欠多少、欠款时间、欠款原因
- **支持功能**：部分还款、全额还款、记录还款时间
- **自动生成**：对账单
- **一键查看**：今日欠款、本月欠款、所有未结清

### 4. 库存管理
- **实时库存**：什么货、剩多少、成本多少
- **筛选功能**：按品种、等级、入库时间筛选
- **库存预警**：可选功能

### 5. 客户管理
- **客户信息**：客户姓名、电话、地址、备注
- **查看历史**：买过几次、总金额、欠款总额
- **客户分类**：优质客户 / 一般客户 / 欠款客户

### 6. 单据与打印
- **单据类型**：出货单、对账单、入库单
- **打印功能**：支持手机连接打印机直接打印

### 7. 数据统计
- **时间维度**：今日 / 本月 / 全年：入库、出库、利润、欠款统计
- **图表展示**：销量排行、利润趋势
- **数据导出**：导出 Excel 表格

### 8. 多账号与权限
- **老板账号**：查看全部
- **员工账号**：只能开单，不能看利润、不能删数据
- **安全控制**：防止员工乱改、乱删

### 9. 数据安全
- **数据备份**：数据云端备份
- **换手机不丢数据**
- **登录方式**：支持账号密码登录

## 四、客户价值

- 不用本子、不用计算器，手机一秒开单
- 欠款再也不会忘、不会乱、不会扯皮
- 库存实时清楚，不用盘点
- 客户信息永久保存，不怕换微信
