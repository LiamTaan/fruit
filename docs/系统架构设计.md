## дёҖгҖҒжҠҖжңҜйҖүеһӢ

### 1. еүҚз«ҜжҠҖжңҜж ҲпјҲеҫ®дҝЎе°ҸзЁӢеәҸпјү

- **зҠ¶жҖҒз®ЎзҗҶ**пјҡе°ҸзЁӢеәҸ globalData жҲ– PiniaпјҲеҰӮйңҖеӨҚжқӮзҠ¶жҖҒпјү
- **еӣҫиЎЁеә“**пјҡECharts for еҫ®дҝЎе°ҸзЁӢеәҸпјҲж•°жҚ®з»ҹи®ЎеӣҫиЎЁпјү

### 2. еҗҺз«ҜжҠҖжңҜж ҲпјҲSpring Boot еҚ•дҪ“жһ¶жһ„пјү
- **ж ёеҝғжЎҶжһ¶**пјҡSpring Boot 2.7.18

- **ж•°жҚ®еә“**пјҡMySQL 8.0пјҲе…ізі»еһӢж•°жҚ®пјү
- **зј“еӯҳ**пјҡRedis 7.xпјҲзј“еӯҳгҖҒдјҡиҜқпјү
- **е®үе…ЁжЎҶжһ¶**пјҡSpring Security + JWTпјҲjjwt 0.11.5пјү
- **APIж–ҮжЎЈ**пјҡKnife4j 3.0.3пјҲSwaggerеўһејәзүҲпјү
- **APIеҚҸи®®**пјҡRESTful API
